## Getting Started

### Clone the repository and cd into the folder

Download and unzip the repository ZIP-file: https://github.com/StefanNieuwenhuis/dpg-media-refactor-assignment/archive/refs/heads/main.zip

OR `clone` the repository:

```bash
git clone https://github.com/StefanNieuwenhuis/dpg-media-refactor-assignment.git
```

### CD into the directory

```bash
cd dpg-media-refactor-assignment
```

### Initialize a Python Virtual Environemnt (optional, but recommended)

Create AND activate a virtual environment for this project. See the [documentation](https://docs.python.org/3/library/venv.html).

### Install dependencies

```bash
python -m pip install -r requirements.txt
```

### Run unit tests

```bash
pytest --verbose
```
